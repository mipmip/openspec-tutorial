A beginner-friendly, complete introductory tutorial for OpenSpec. Learn how to collaborate with AI for spec-driven development through hands-on examples.

## 📚 About This Project

OpenSpec is a **Spec-Driven Development framework for AI coding assistants**. It helps you align requirements, design, and tasks with AI before writing code, so you avoid the dreaded "that's not what I wanted" moment.

### Core Philosophy

```
→ fluid not rigid          → Flexible, not rigid
→ iterative not waterfall  → Iterative, not waterfall
→ easy not complex         → Simple, not complex
→ built for brownfield     → Works with existing projects
→ scalable                 → From personal projects to enterprise scale
```

## 🚀 Quick Start

### Install OpenSpec

```bash
# Requires Node.js 20.19.0+
npm install -g @fission-ai/openspec@latest
```

### Initialize a Project

```bash
# Navigate to your project directory
cd your-project

# Initialize OpenSpec
openspec init
```

### Your First Change

```bash
# Option 1: Create directly with the CLI
openspec new change add-user-login

# Option 2: Tell your AI tool
/opsx:propose "Add user login feature"
```

The AI will automatically create:
```
openspec/changes/add-user-login/
├── proposal.md      # Proposal: why we're doing this
├── specs/           # Specs: requirement details
│   └── requirements.md
├── design.md        # Design: technical approach
└── tasks.md         # Tasks: implementation checklist
```

### Start Implementing

Tell your AI tool:

```
/opsx:apply
```

The AI will implement the code following tasks.md automatically.
